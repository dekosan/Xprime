## Xprime Code Editor for macOS
Edit your PPL or PPL+ code for the HP Prime.

### Lauching the HP Applications

### Run
Preferences let you set library and include path locations, and choose whether to use macOS or Wine for the HP Prime Virtual Calculator.

<img width="50%" src="https://raw.githubusercontent.com/Insoft-UK/PrimeSDK/main/assets/screenshots/Preferences.png" />

### Run
The PRGM+ source code is preprocessed into a proper HPPRGM G1 file, which is automatically transferred to the HP Prime Virtual Calculator for execution and testing.

<img width="25%" src="https://raw.githubusercontent.com/Insoft-UK/PrimeSDK/main/assets/screenshots/Run.png" />

### Templates
This section provides template snippets for common programming constructs (e.g., for loops, if statements) that you can insert directly into your code.
